# TaskFlow Pro — Brand Voice & Communication Style Guide

## Core Brand Personality
- **Friendly** — Warm and approachable, never robotic
- **Professional** — Competent and reliable, never sloppy
- **Empathetic** — We understand frustration and validate feelings
- **Clear** — Simple language, no jargon unless the customer uses it first
- **Solution-oriented** — Always guide toward resolution

## General Rules
1. Use the customer's name when available
2. Never blame the customer
3. Apologize sincerely when we're at fault — don't over-apologize
4. Use active voice ("You can reset your password by..." not "Your password can be reset by...")
5. Avoid filler phrases ("I hope this email finds you well", "Please do not hesitate to...")
6. End with a clear next step or open offer to help

## Channel-Specific Formatting

### Email
- **Tone:** Professional, warm, thorough
- **Length:** Medium to long — cover the topic fully
- **Structure:**
  - Greeting with name
  - Acknowledge issue/question
  - Detailed answer with steps
  - Links to relevant help articles
  - Sign-off with agent name and support team
- **Example sign-off:** "Best regards, [Agent Name] — TaskFlow Pro Support"

### WhatsApp
- **Tone:** Friendly, concise, conversational
- **Length:** Short — 2-4 sentences per message, break long answers into multiple messages
- **Structure:**
  - Quick greeting (no formal "Dear")
  - Direct answer
  - Use emojis sparingly (1-2 max): ✅, 👍, 📎
  - No formal sign-off needed
- **Formatting:** Use WhatsApp formatting (*bold*, _italic_) for emphasis

### Web Form / Live Chat
- **Tone:** Semi-formal, helpful, responsive
- **Length:** Medium — concise but complete
- **Structure:**
  - Brief greeting
  - Direct answer with steps
  - Offer follow-up help
- **Formatting:** Use bullet points, keep paragraphs short

## Tone Adjustments by Situation

| Situation           | Tone Adjustment                                   |
|---------------------|---------------------------------------------------|
| Happy customer      | Match their energy, thank them                    |
| Confused customer   | Be extra clear, use step-by-step format           |
| Frustrated customer | Lead with empathy, acknowledge frustration first  |
| Angry customer      | Stay calm, validate feelings, escalate if needed  |
| Technical user      | Match technical depth, skip basics                |
| New user            | Be welcoming, provide extra context               |

## Words to Use / Avoid

| Use                        | Avoid                          |
|----------------------------|--------------------------------|
| "I understand"             | "Obviously"                    |
| "Let me help with that"    | "You should have"              |
| "Great question"           | "As per my last message"       |
| "Here's what you can do"   | "That's not possible"          |
| "I'll look into this"      | "That's not my department"     |
| "Thank you for your patience" | "Calm down"                 |
