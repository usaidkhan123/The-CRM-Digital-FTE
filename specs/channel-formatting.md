# Channel-Specific Response Formatting

## Email
- **Tone:** Professional, warm, thorough
- **Length:** 150–400 words
- **Structure:**
  ```
  Hi {name},

  {Acknowledge issue/question}

  {Detailed answer with numbered steps if applicable}

  {Link to help article if relevant}

  {Clear next step or open offer}

  Best regards,
  TaskFlow Pro Support
  ```
- **Rules:**
  - Always use customer name
  - Include step-by-step instructions when applicable
  - Reference help articles with URLs
  - Professional sign-off
  - Full sentences, proper grammar

## WhatsApp
- **Tone:** Friendly, concise, conversational
- **Length:** 30–100 words per message
- **Structure:**
  ```
  Hi {name}! 👋

  {Direct answer in 2-3 sentences}

  {Quick follow-up or offer to help}
  ```
- **Rules:**
  - Max 1-2 emojis (✅, 👍, 📎, 👋)
  - Use WhatsApp formatting: *bold* for emphasis
  - No formal greeting/sign-off
  - Break long answers into multiple short messages
  - Okay to use contractions ("you're", "can't", "it's")
  - Match informal tone if customer uses it

## Web Form / Live Chat
- **Tone:** Semi-formal, helpful, efficient
- **Length:** 80–200 words
- **Structure:**
  ```
  Hi {name},

  {Brief acknowledgment}

  {Answer with bullet points or short steps}

  Let me know if you need anything else!
  ```
- **Rules:**
  - Use bullet points for multi-step answers
  - Keep paragraphs to 2-3 sentences
  - Friendly but not overly casual
  - Include relevant links when helpful

## Escalation Response Formatting

### Email (escalation)
```
Hi {name},

Thank you for reaching out. I understand {empathy statement}.

I've created a priority ticket (#{ticket_id}) for our specialist team, and they will follow up with you within {SLA timeframe}.

In the meantime, {any interim guidance if applicable}.

Best regards,
TaskFlow Pro Support
```

### WhatsApp (escalation)
```
Hi {name}, I understand this is {frustrating/urgent/important}.

I've flagged this to our specialist team — ticket #{ticket_id}. They'll reach out to you shortly. 🙏
```

### Web Form (escalation)
```
Hi {name},

I understand your concern. I've escalated this to our specialist team with priority.

- Ticket: #{ticket_id}
- Expected response: {SLA timeframe}

We'll follow up as soon as possible.
```
